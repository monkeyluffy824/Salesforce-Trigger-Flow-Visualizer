Welcome to Salesforce Object Trigger Flow Visualizer

This is an browser extension which can be used to visulaize the visualize order of flows and trigger execution, in diagrams using mermaid.js

Connect to Salesforce

How To Use

  • Login to your org and once logged in open this extension.
  • You can see the currently loggged in Orgs, in right side menu.
  • Select the org and click on connect to salesforce button.
  • Now you can see all triggerable objects present in that org that can be visualized.
  • To search enter text in the search bar and press ENTER.
